# Progress log

Append-only. One terse line per meaningful step, newest at the bottom. This is what a fresh context
reads (with `git log`) to see what just happened. Don't editorialize — facts and pointers.

Format: `- YYYY-MM-DD <what happened> (<commit/evidence ref>)`
